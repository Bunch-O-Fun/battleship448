How to Launch the Project:
1) Download the project to your device.
2) Open your Terminal and navigate to the project directory.
3) Type "make" into your Terminal.
4) Type "./bship" to run the program.

How to Play:
This battleship game takes place entirely in your Terminal and relies on your simple inputs.
You will be prompted to enter values for determining how many ships you wish to play with,
whether or not you wish to play against another person or the computer, how you would like
to place your ships, and what coordinates you wish to fire upon.

After you type an input into the terminal, you will need to press the [Enter] key for
your input to be processed by the game.

The game will specify what range of inputs it will accept from you each time it prompts
you for an input. If your input is invalid, the game will simply prompt you for an input
again until you enter a valid input.

Once the game has finished, the program will stop, and if you wish to play again, you will
need to type "./bship" into the Terminal to run the program again.

Have fun!
